# RECAP - DOCKER CONTAINER

***COMMAND LINE***
* DOCKER RUN [OPTIONS] IMAGE [COMMAND] [ARG...]

## OPTIONS
* -d ( detach, chạy nền )
* -it ( interactive + tty, dùng cho terminal)
* --name <container_name> ( đặt tên cho container )
* -p host_port:container_port ( Map Port )
* -v host_port:container_port ( Map Volumes )
