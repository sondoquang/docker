# DOCKER REGISTRY

## Docker Registry:

- Nơi lưu trữ và chia sẻ Docker Images, vd [Docker Hub](https://hub.docker.com/)
- Cho phép push image sau khi buil và pull image về local để chạy container

## Một số loại Docker Registy:

1. Public Registry:
   - Docker Hub ( mặc định )
   - Dễ dàng sử dụng, miễn phí cho image công khai
2. Private Registry:
   - Lưu trữ image nội bộ, bảo mật hơn.
   - Các dịch vụ cloud đều có
3. Tự triển khai Registry
